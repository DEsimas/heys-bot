## Using
This bot can't be added on your server
## Hosting
clone repository
```
git clone https://github.com/DEsimas/heys-bot
```
add .env file with all neded variables

install dependencies
```
npm i
```
build project
```
npm run build
```
start bot
```
npm run start
```
